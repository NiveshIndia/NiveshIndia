# NiveshIndia 🏠

**India's Premier NRI Property Investment Platform**

Built for NRIs looking to invest in Indian real estate from abroad.

## Features
- 🏠 Browse verified properties across Mumbai, Bangalore, Hyderabad, Delhi NCR, Chennai
- 💰 ROI Calculator with USD/INR conversion
- 📊 Market insights and city-wise trends
- ❤️ Watchlist to save favourite properties
- 🔍 Filter by city, property type, yield
- 📱 Mobile-first design

## Tech Stack
- React 18
- Deployed on Vercel

## Getting Started
npm install
npm start
